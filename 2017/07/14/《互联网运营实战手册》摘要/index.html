《互联网运营实战手册》摘要

发布 : 2017-07-14 分类 : 读书笔记 浏览 :

运营的分类

  • 用户运营：负责增加新用户
  • 新媒体运营：负责公司的微博、微信公众号的管理和内容发布
  • 活动运营：策划一个线上活动或地推活动
  • 社群运营：维护社群
  • 数据运营：负责推广网站(产品)并保证产品用户的留存率、活跃度、转化率
  • 产品运营
  • 内容运营：维护社区用户发布内容、管理社区工作

运营是什么

  • 拉新：为产品带来新用户或流量
  • 留存：让用户留下来真正去使用产品
  • 激活：通过一些方式方法与用户发生连接关系，提高用户的长期活跃度
