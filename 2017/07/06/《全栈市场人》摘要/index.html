《全栈市场人》摘要

发布 : 2017-07-06 分类 : 读书笔记 浏览 :

适合做互联网的人

  • (1).超强的学习和借鉴能力
  • (2).熟稔任性，略通传播学和社会学
  • (3).极强的个人魅力
  • (4).过硬的技术(技术岗，程序员类)
  • (5).整合资源的能力

大公司打仗的时候，拼的是你的”多”和”好”，初创公司探路的时候，比的就是你的”快”和”省”

互联网主要工种

  • 技术岗
    • 前端
    • 后端
    • 测试
    • 运维
  • 销售岗
  • 职能岗
  • 市场岗
    • 市场部对外，负责拉新[引进新用户]
    • 公关组(官方发言人)
      • 大多从传统媒体转行来，一般和各大门户网站有较深的合作关系，能够保证正面新闻的第一时间铺出，有会撰写各种关于公司或者行业的新闻稿，同时还负责邀请各家媒体对公司进行采访和报道，以提高公司的曝光度、美誉度以及公信力
    • 品牌组(塑造企业人话和企业形象)
      • 大多出身4A广告公司，有平面广告经验、矫情的创意能力，通过海报、广告语，掌握公司品牌的格调高低，也负责举办行业内有较大公信力的盛会和演讲论坛
      • 日常工作就是发布调查问卷，搜集群众的品牌印象，去验证品牌塑造效果
    • 新媒体组(产出内容，购买媒介)[企业看重市场部的新宠]
      • 负责撰写及编辑优质内容，一些人负责对接供应商，实现渠道资源购买，而打下手做发布，物料整理等基础工作则交由实习生来做
      • 终日以涨粉、十万加的阅读量、互推、做活动为目标
    • SEM组(搜索引擎优化和百度品牌专区购买)
      • SEO搜索引擎优化，通过拓展关键词来建设相应的页面，提升网站自然搜搜流量;同时需要负责网站结构、站点内容、外链等优化工作，跟进推进进度，发现存在的问题并制定改进方案
    • BD组(商务拓展)
      • 不怎么涉及钱的销售
      • 以自身长袖善舞的沟通能力，在对公司产品充分了解的基础上，那合适的资源去换取其他公司的资源，以达到资源置换的效果
    • 移动端APP组(App在各大软件商店的上架和推广)
      • IOS端：专门找一个做ASO的做APP Store推广
      • Android端：各大应用市场申请首发、专题、有奖活动、直接购买广告位(cpt即时长付费[为曝光]，cpd即按天付费[为效果])/关键词覆盖/热词采纳
  • 运营岗
    • 运营部对内，负责留存和转化(留存指使用户留在软件内保持使用，转化指将用户数转换成实际的交易量或者其他商业目的的最终行为)
    • 数据运营
    • 活动运营
    • 用户运营
    • 商家运营
    • 品类运营
    • 游戏运营
    • 网店运营
    • 内容运营
  • 产品岗
  • 设计岗
  • 永远不要说”我不知道”
  • 具备职业精神
  • 学会写邮件
    • 使用Execl一目了然
  • 向上级汇报
  • 不要越俎代庖
  • 疯狂学习，低调处事，锻炼身体
  • 你不能指望面试官一眼看出你靠谱
  • 进入职场，你是来工作、成就事业的，不是
  • 用数据说话

工作过程中遇到的专有名词，一定要不懂就问，不懂就查

互联网公司工作常用术语

  • CC：抄送
  • PV：Page View，页面浏览量
  • UV：Unique Visitor，网站独立访客
  • 996：早9点上班，晚9点下班，一周工作六天
  • KPI：Key Performance Indicator，关键绩效指标法，即工作业绩
  • ROI：Return on Investment，投资回报率
  • CPM：Cost Per Mille，每千人成本
  • DAU：Daily Active User，日活跃用户数量
  • PCU：Peak Concurrent Users，最高同时在线用户人数
  • MAU：Monthly Active Users，月活跃用户人数
  • CPC：Cost Per Click，每次点击付费
  • FYI：For Your Information，仅供参考
  • UGC：User Generated Content，用户原创内容
  • 拖库：黑客盗走了整个数据库的信息
  • 撞库：由于大多数用户在各大网站上所使用的用户名和密码是相同的，所以黑马会将得到的数据在其他网站上进行尝试登录
  • 审查元素：查看网页源代码
  • GA：谷歌统计
  • 期权：大饼
  • 地推：地面推广，在路边做各种拉新用户的行为
  • 黑公关：专门写公司负面新闻以要挟对方付费删稿的恶意敲诈行为
  • 比稿：同一个需求会让不同的人同时写稿，先筛选而后确定，并付与作者稿费
  • 车马费：邀请记者来参加发布会的时候，会以车马费的名义给予记者现金，让记者能心甘情愿地发稿
  • 软文：伪造成费明显广告形式的植入性广告的掩体文章
  • 复盘：项目结束的回顾总结
  • IMI：数字新媒体
  • SMART：制定计划的通用规则
    • S(Specific)绩效指标必须是具体的
    • M(Measurable)绩效指标必须是可以衡量的
    • A(Attainable)绩效指标必须是可以达到的
    • R(Relevant)绩效结果和目标是有相关性的
    • T(Timed-bound)绩效指标必须具有明确的截止期限

    • 用户画像

    • 哪些公司决不能合作
    • 哪些公司合作有危害
  • 梳理公司资源库
